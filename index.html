ThriftPool¶

ThriftPool is an application server for Thrift services. It should create needed sockets, start workers, serve requests from client and log them. It use pyuv as wrapper for libuv and ØMQ for load balancing between workers. Request processing written in Cython.

Note

Currently it support only Thrift Framed protocol.

Key features:

  • Fast request processing (~3000 rps);
  • Compatibility with gevent through monkey patching;
  • Support request logging;
